Reseña documental Código Linux



En este documental se puede apreciar el surgimiento del nuevo sistema operativo Linux, el cual es totalmente gratuito y editable para el usuario, todo esto surge en protesta a la forma de comercio y uso de los sistemas operativos mas famosos los cuales eran privados y no permitían la alteración o desarrollo externo de estos sistemas.

Los beneficios de generar un sistema operativo libre son tanto para la empresa creadora, como para los usuarios, ya que al liberar el sistema operativo se amplia la mano de obra de manera gratuita y sele dan mas libertades al usuario de crear un sistema operativo el cual pueda usar de manera mucho más libre y fácil.

Como resultado de los hechos mostrados en el documental, hoy en día se cuenta con mas de 300 distribuciones o tipos de este sistema operativo que no solo la empresa Linux si no también otros desarrolladores particulares han producido para todos los usuarios.
